# LifeWeeksBot 🧠

## 🔥 About

Telegram bot that tracks user's life progress in weeks and sends personalized weekly insights.

## ⚙️ Tech Stack

* Python
* aiogram
* PostgreSQL
* APScheduler
* Docker

## 🚀 Features

* User registration
* Date of birth tracking
* Weekly notifications
* Life expectancy calculations
* Personalized messages

## 🧠 Architecture

* Async bot (aiogram)
* Scheduler for weekly tasks
* Database for user data

## ▶️ Run

```bash
python -m venv venv
pip install -r requirements.txt

python main.py
```
